Code de Conduite
================

Notre Engagement
----------------

Nous nous engageons à faire de la participation à notre projet une expérience exempte de harcèlement pour tous.

Standards de Comportement
--------------------------

Comportements encouragés :

* ✅ Utiliser un langage accueillant et inclusif
* ✅ Respecter les différents points de vue et expériences
* ✅ Accepter gracieusement les critiques constructives
* ✅ Se concentrer sur ce qui est le mieux pour la communauté
* ✅ Faire preuve d'empathie envers les autres membres

Comportements inacceptables :

* ❌ Langage ou images sexualisés
* ❌ Trolling ou commentaires insultants
* ❌ Harcèlement public ou privé
* ❌ Publication d'informations privées sans permission
* ❌ Toute conduite inappropriée dans un cadre professionnel

Application
-----------

Les instances de comportement abusif peuvent être signalées en contactant l'équipe du projet.

Pour plus de détails, consultez le `Code de Conduite complet <https://github.com/USERNAME/projet1/blob/main/.github/CODE_OF_CONDUCT.md>`_.
